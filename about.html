About This Project

MusicFlow is a student web project created in 2024 to explore modern web technologies and user interface design for a music streaming experience. The goal is to provide a simple, clean, and interactive platform for browsing playlists and playing tracks.

This project demonstrates dynamic playlist management, a working audio player, light/dark mode, and responsive design. It is not a commercial service, but a learning exercise in building a music-themed web application.

Features

Dynamic Playlists

Create, view, and manage playlists with a modern UI. Playlists are rendered dynamically from JavaScript data.

Audio Player

Play tracks, manage a queue, and use shuffle/repeat features. The player is built with standard web audio APIs.

Light/Dark Mode

Switch between light and dark themes. The selected mode is saved for a consistent experience.

Responsive Design

Enjoy a layout that adapts to desktop and mobile screens using Bootstrap and custom CSS.
